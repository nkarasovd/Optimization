Критерий номер 3.

Остановка происходит, когда разность значений функции в двух соседних точках улучшения меньше epsilon. |f(x_{k+j}) - f(x_{k})| < epsilon Как только |f(options.get_x_k_Plus_j()) - f(options.get_x_kj())| < epsilon, то сразу заканчиваем поиск.
